# PROYECTO FINAL CODERHOUSE

## Partes del proyecto
Consta de un backend desarrollado con django y django restframework, y el frontend con flet

## Comandos para la ejecucion.

### backend.
- pip install -r requirements.txt
- cd api_blog
- python manage.py runserver

### frontend.
- pip install -r requirements.txt
- python main.py
